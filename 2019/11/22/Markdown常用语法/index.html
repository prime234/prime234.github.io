推荐的编辑器

  • VSCode (下载插件就行，强推)

  • MarkdownPad 2(windows)

  • Typora

    1. 支持6个级别的标题大小

    #### 四级标题
##### 五级标题
###### 六级标题

    2. 字体

    **这里想加粗**
*这里想用斜体*
***这里加粗斜体***
<u>这里想用下划线</u>
~~这里想用删除线~~

    效果：

    这里想加粗

    这里想用斜体

    这里加粗斜体

    这里想用下划线

    这里想用删除线

    注：如果在段落中想给文字换颜色，换字体字号或居中，使用内嵌HTML的方法实现

3. 引用

文字前加”>”

>这里想引用

4. 列表

  • 分有序列表、无序列表还可以进行列表嵌套（开头加空格）
  • 有序列表是数字后加”.”和空格，无序列表在文字前加”-/+/*”的任何一个和空格。

效果：

  • 这里是无序列表1

  • 这里是无序列表3
  1. 这里是一级有序列表
    • 这里是二级无序列表（开头打两个空格）
