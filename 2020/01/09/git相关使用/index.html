使用git将项目上传到Github

首先你需要有github账号

  • github官网

安装git工具

  • git官网

  • 参考资料：如何在window下安装git(by俊雨廷休）

进入Github首页，点击New repository新建一个项目

填写相应信息后点击create即可,点击Clone or dowload会出现一个地址，copy这个地址备用。

Repository name: 仓库名称
Description(可选): 仓库描述介绍
Public, Private : 仓库权限（公开共享，私有或指定合作者）
Initialize this repository with a README: 添加一个README.md
gitignore: 不需要进行版本管理的仓库类型，对应生成文件.gitignore
license: 证书类型，对应生成文件LICENSE

接下来本地操作，右键你的本地项目选择Git Bash Here,进入以下界面。

关键环节：输入以下代码

git clone git@github.com:prime234/Pandas-.git（地址替换你刚刚复制的GitHub仓库地址）//把github上面的仓库克隆到本地

这时就在本地创建好了GitHub仓库的文件夹，将你要上传的项目复制或移动在这个文件夹目录下

输入命令
cd Pandas- //进入文件夹

继续命令

git add . （注：别忘记后面的.，此操作是把Test文件夹下面的文件都添加进来）
git commit -m “提交信息” （注：“提交信息”里面换成你需要的说明）
git push -u origin master （注：此操作目的是把本地仓库push到github上面，此步骤需要你输入帐号和密码）

删除Github上仓库里的文件

删除已有仓库

进入需要删除的仓库里，找到仓库设置”setting”,然后在settings最底部”Danger Zone”点击”Delete this repository”即可。

删除GitHub中某文件或文件夹

  1. 桌面新建一个文件夹delete

  2. 新建文件夹里右键git bash here打开终端，执行git init初始化仓库

  3. 复制github上要删除的仓库地址，终端输入git clone <仓库地址>

  4. 会多一个文件夹，打开该文件夹右键git bush here打开新的终端

  5. 在终端输入dir查看此文件夹下文件目录

  6. 删除本地仓库文件git rm <要删除的文件名>或文件夹git rm -r <要删除的文件夹名>

  7. 删除以后提交本次修改`git commit -m “提交信息”

  8. 把删除文件夹回复最新的版本git checkout

  9. 将本地内容推送远程仓库git push

  10. 刷新仓库

    finish！！！
